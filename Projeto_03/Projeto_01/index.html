Olá mundo é um famoso programa de computador que imprime "olá mundo" (ou "hello world!"), usualmente seguido de uma quebra de linha, com algumas variações como inexistência do ponto de exclamação e letras em minúsculas, no dispositivo de saída.

É utilizado como um teste ou como um exemplo de código de uma linguagem.
